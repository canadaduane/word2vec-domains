A Word2Vec API Example App

Domain Topic Words:

Domain Thesaurus is an example showing how to use the Word2Vec API:

When you provide an English word to this API, it will give you a list of related English words scored by "distance" in meaning. Uses natural language processing (word2vec, GLoVe) and lets you select any of 3 large corpuses: Wikipedia, Common Crawl (the internet), and Gutenberg (19th Century Books).
